## üõ†Ô∏è How to Export Configuration Files

To include configuration scripts in your report or repository, follow these steps in **Cisco Packet Tracer**:

1.  **Open the CLI:** Click on the specific Router or Switch.
2.  **View Config:** Enter Privileged EXEC mode and type:
    ```bash
    enable
    copy running-config startup-config
    ```
    Press Enter again when asked, then enter
    ```
    show running-config
    ```
4.  **Copy Text:** Scroll through the output, select the text (configuration commands), and copy it. *(Press Spacebar to scroll to the end)*
5.  **Save File:** Paste the content into a text editor (like Notepad or VS Code) and save it as `DeviceName_Config.txt`.
