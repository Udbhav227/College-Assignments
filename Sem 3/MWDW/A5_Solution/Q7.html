Lists in HTML

HTML provides three types of lists: ordered, unordered, and definition lists. Each has its own use cases for organizing information.

Ordered List
Displays items in a numbered order.
Unordered List
Displays items with bullet points.
Definition List
Defines terms and their descriptions.

CSS Selectors

CSS selectors are used to select HTML elements to apply styles.

Element Selector
Selects elements based on their tag name.
Class Selector
Selects elements based on their class.
ID Selector
Selects an element based on its ID.

CSS Pseudo-Classes

Pseudo-classes are used to define the special state of an element.

:hover
Applies styles when an element is hovered over.
:active
Applies styles when an element is clicked.
:focus
Applies styles when an element is focused.
